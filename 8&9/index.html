1-HTTP Protocol: A set of rules used to transmit hypertext and web pages over a network. 2-HTTPS protocol: It is the same as http protocol, except that it is encrypted correctly by the transfer layer security protocol. 3-FTP Protocol: The rules for transferring files over the Internet. 4-SMTP protocol: rules for e-mail transfers. 5-NCP Protocol: The core rules. 1-پروتکل HTTP : مجموعه ای از قوانین که برای انتقال ابر متن ها و صفحات وب در شبکه به کار می رود. 2-پروتکل HTTPS : همان پروتکل http است اما با این تفاوت که توسط پروتکل امنیت لایه انتقال ، به طور صحیح رمزگذاری می شود. 3-پروتکل FTP : قوانین انتقال فایل در اینترنت است. 4-پروتکل SMTP : قوانین برای انتقالات پست الکترونیک است. 5-پروتکل NCP : قوانین هسته مرکزی است.
